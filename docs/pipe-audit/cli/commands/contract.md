# `pipa contract`

Manage data contracts.

- List contracts:
  pipa contract list

- Show contract details:
  pipa contract show \<CONTRACT_NAME\>

- Validate contract syntax:
  pipa contract validate \<PATH_TO_CONTRACT.toml\>
