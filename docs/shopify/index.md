# Shopify Apps

Purpose-built apps for merchants. Simple tools that solve real problems.

---

## Philosophy

Same Lokryn DNA, different channel. These apps follow our core principles:

- **Does one thing well** — No feature bloat
- **Fair pricing** — Usage-based or flat fee, no enterprise minimums
- **Simple setup** — Working in minutes, not days

---

## Available Apps

### [Affiliate Ping](/docs/shopify/affiliate-ping)

Real-time notifications when affiliate and influencer discount codes drive sales.

- **Instant alerts** — Know within seconds when a code converts
- **Email & Slack** — Your choice of notification channel
- **Track any code** — Works with existing Shopify discount codes
- **Unlimited codes** — Track as many as you need

**Install from:** [Shopify App Store](https://apps.shopify.com/affiliate-ping)
